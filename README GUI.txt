This code generate a GUI for the Sitrineo's FPGA device.
Thanks to the dedicate buttons you can:
- initialise (reset button)
- configurate (config button)
- start (star button)
- acquisition (button which open a popup and where the user can chose the parameters such as the run number, the number of events and the trigger) 

There is also a save button to export the data from the FPGA to the PC and create a folder for it.

Finally, the user can also use the button TAF to open the GUI dedicated to TAF for the analysis of the data.
